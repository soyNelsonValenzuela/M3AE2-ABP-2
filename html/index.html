Proyecto BEM

La metodología BEM (Bloque, Elemento, Modificador) es una metodología para nombrar clases CSS que facilita la escritura de código estructurado, claro y escalable. Su objetivo principal es mejorar la organización del HTML y CSS, haciendo que los estilos sean más fáciles de mantener y evitando conflictos entre clases. En BEM, un bloque representa un componente independiente de la interfaz (como header, form o card), un elemento es una parte del bloque que no tiene sentido por sí solo (como card__title o form__input), y un modificador define una versión o estado diferente de un bloque o elemento (como button--primary o nav__link--active). Esta convención ayuda a mantener una jerarquía clara, especialmente en proyectos grandes o en equipos donde múltiples personas trabajan sobre el mismo código. Implementar BEM permite mejorar la legibilidad del código, facilita su reutilización y reduce errores al aplicar estilos.

Esta página web fue desarrollada aplicando la metodología BEM para organizar y modularizar los estilos CSS de manera clara, escalable y siguiendo buenas prácticas de desarrollo frontend.

Se cumple con los requerimientos de crear una estructura de carpetas ordenada (html/ e css/), incorporando un header con logo y navegación, un contenido principal con un artículo, una galería de imágenes y un formulario de contacto, además de un footer con información de contacto y enlaces a redes sociales.

La estructura HTML y las clases CSS siguen estrictamente el formato BEM, separando claramente los bloques (header, article, form, footer) y sus elementos (como header__logo, form__input, gallery__image, etc.).

Se implementaron modificadores para adaptar el comportamiento visual de algunos elementos, como los nav__link--disabled para enlaces inactivos en el menú de navegación, los nav__link--active para enlaces activos en el menú dee navegación y los footer__icon--facebook, --twitter y --instagram para dar estilos personalizados a los íconos de redes sociales.

La interfaz es funcional y presenta una estructura visual coherente, con una base de responsividad para asegurar su correcta visualización en distintos tamaños de pantalla.
